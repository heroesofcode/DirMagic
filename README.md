# DirMagic

DirMagic is a library written in Python to brute force directories and DNS

### How to install?
```
git clone https://github.com/heroesofcode/DirMagic.git
```

### How to use?

```
cd DirMagic
pip install -r requirements.txt
python3 dir.py
```
